# Pasta de Recursos

Esta pasta deve conter os binários e modelos necessários para a aplicação funcionar.

1.  **Binários:**
    *   Coloque os executáveis `ffmpeg` e `ffprobe` aqui.
    *   Baixe-os do site oficial do FFmpeg.

2.  **Modelos Pré-treinados:**
    *   Crie uma pasta chamada `pretrained_models`.
    *   Dentro dela, coloque a pasta `2stems` (e outras, se necessário) contendo os arquivos de modelo do Spleeter.
